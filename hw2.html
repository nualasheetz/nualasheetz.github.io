Here is part 2 of Nuala's homework with a form that has:

  • three input boxes for the user to type three integers (numbers)
  • a button to process the data (numbers) typed by the user
  • a button to clear the form
  • a textarea to display the results of the process (or it can be a div or paragraph element)

JavaScript code will:

  • receive the data from those three input boxes for the three integers after the user click on the button to process the data
  • calculate the sum, the average, and the product of the three integers
  • calculate the smallest and the largest of the three integers typed
  • present the results of these calculations in the text area
  • present an error message in the textarea if the user does not type NUMBERS in the input boxes
  • use jQuery that will be triggered by a button that you will include in the form and when the user clicks on that button, you should fade the results you presented in the textarea.
